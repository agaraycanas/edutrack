# EduTrack - Contexto del Proyecto

Este fichero sirve como "memoria" sincronizada para que cualquier instancia de Antigravity o IA pueda retomar el proyecto desde cualquier ordenador, conociendo su estado y arquitectura.

## Objetivo
Migración de una aplicación legacy ("Seguiprog", hecha en PHP y XML) a una moderna Single Page Application (SPA). La aplicación sirve para que los profesores hagan un seguimiento del progreso de sus programaciones didácticas (temas impartidos, desviaciones en horas, etc.) y para que la jefatura de estudios tenga informes de seguimiento global.

## Arquitectura y Stack Tecnológico
- **Frontend**: React (creado con Vite). SPA clásica.
- **Estilos**: Vanilla CSS con un enfoque en diseño moderno (Dark Mode, UI estilo corporativo/premium).
- **Backend/Base de Datos**: Firebase Firestore (abandonamos los XML).
- **Autenticación**: Firebase Auth (Google Auth para los profesores).
